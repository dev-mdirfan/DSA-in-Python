# 1. Find all subsets

Given a set of positive integers, find all its subsets.

__Examples:__

```yml
Input: array = {1, 2, 3}
Output: [[], [1], [1, 2], [1, 2, 3], [1, 3], [2], [2, 3], [3]]
```

```yml
Input: [1, 2]
Output: [[], [1], [2], [1, 2]]
```

__Note:__ returns the list of list containing the subsets of the given set of numbers in lexicographical order.
